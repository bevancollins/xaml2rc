# xaml2rc

Generates a win32 resource file (.rc) from a XAML file (.xaml). This allows you to design a user interface in the VisualStudio XAML designer which is superior to the dialog resource editor.

## Usage

    xaml2rc <input path> <output path>

## Dependencies

* [pugixml](https://pugixml.org/)
* [yoga](https://www.yogalayout.dev/)
* [wil](https://github.com/microsoft/wil)
